## API Integration

### Backend Endpoints Used

1. **POST /api/urls**
   - Save current URL
   - Headers: X-API-Key, Content-Type
   - Body: { url, title, faviconUrl }
   - Response: SavedUrl object

2. **GET /api/urls**
   - Get saved URLs
   - Headers: X-API-Key
   - Query: limit, lastKey
   - Response: { urls[], hasMore, lastKey }

- Configuration validation
- Proper error handling
- TypeScript types
- Query parameter building
